# Python Fundamentals & Software Engineering Assignments

This repository contains a structured collection of 9 Python projects developed during my academic studies. The projects demonstrate a clear progression from basic scripting and mathematical logic to advanced programming concepts such as modular design, exception handling, and file text-processing.

## Project Portfolio Overview

| # | Project Name | Directory / File Name | Technical Description | Core Concepts Covered |
| :---: | :--- | :--- | :--- | :--- |
| **1** | Conditional Interest Simulator | `conditional_interest_calculator.py` | A financial simulator implementing compound interest formulas with dynamic interest rate adjustments based on client deposit tiers. | `Conditional Logic`, `Type Casting`, `F-string Formatting` |
| **2** | Dynamic Vineyard Calculator | `vineyard_spacing_calculator.py` | An interactive agricultural spacing calculator that dynamically alters vine and post parameters based on row dimensions within a continuous user loop. | `While Loops`, `Iterative Logic`, `Dynamic Space Allocation` |
| **3** | Yearly Growth Tracker | `yearly_interest_breakdown_simulator.py` | An upgraded financial tool that generates a chronological, formatted multi-year amortization breakdown chart with robust user-input validation. | `For Loops`, `Data Validation`, `Tabular String Alignment` |
| **4** | Static Vineyard Planner | `static_vineyard_spacing_calculator.py` | A single-run engineering calculator designed to execute geometric vineyard spacing equations based on structured length conditions. | `Boundary Condition Testing`, `Mathematical Modeling` |

## Technical Milestones Demonstrated
* **Procedural to Functional Programming:** Transitioned from flat scripts to clean, modular, functional software architectures.
* **Data Automation & Validation:** Implemented automatic file parsing mechanisms combined with custom constraint checkers.
* **Defensive Software Engineering:** Learned to properly isolate runtime crashes using elegant error-trapping blocks to ensure application stability.
